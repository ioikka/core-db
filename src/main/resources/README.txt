Утилита позволяет переносить структуры и данные из СУБД Oracle в СУБД Postgres.

В рамках поставленной задачи использование утилиты предполагается урезанным. 
Метаданные проливаютсся с помощью DBUpdate, а данные переносятся данной утилитой.


Алгоритм конвертанции на данный момент: 

1. executer create_user
2. perform 1
3. настроить утилиту через файл project.properties
4. java -jar core-db-1.0-SNAPSHOT.jar   

Проект собирается maven-ом.
